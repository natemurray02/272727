## How to Play

1. **Create a Table**: Enter your name, choose table size (2-9), blinds, and buy-in
2. **Share the Code**: Give the 6-character room code to your friends
3. **Friends Join**: They enter the code to join your table
4. **Start Game**: Host clicks "Start Game" when ready
